KEY METRICS
================================================================================

Model Size: 3 MB (nano version)
Download Size: 3 MB (one-time, then cached)
Inference Time: 30-50 ms per frame
Frame Rate: 20-35 FPS on CPU
Memory Usage: 300-500 MB during operation
Startup Time: 2-3 seconds (model loading)
Classes Detected: 80 different object types
Accuracy: 75-80% (nano), higher with larger models

COMPARISON:

Your Laptop:   30-50 ms latency, works offline
